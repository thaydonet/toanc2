<h3 id="phep-cong">📖 1. Phép cộng số tự nhiên</h3>

<div class="box-dinh-nghia">

**Định nghĩa:** Phép cộng hai số tự nhiên $a$ và $b$ cho ta một số tự nhiên, gọi là **tổng**, ký hiệu là $a + b$.

- $a, b$: các **số hạng**
- $a + b$: **tổng**

</div>

**Tính chất của phép cộng:**

<div style="background:#f0f9ff;border-left:4px solid #0ea5e9;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

**1. Tính chất giao hoán:**

$a + b = b + a$

Ví dụ: $5 + 7 = 7 + 5 = 12$

**2. Tính chất kết hợp:**

$(a + b) + c = a + (b + c)$

Ví dụ: $(3 + 5) + 7 = 3 + (5 + 7) = 15$

**3. Cộng với số 0:**

$a + 0 = 0 + a = a$

Ví dụ: $25 + 0 = 25$

</div>

**Ví dụ 1:** Tính nhanh

a) $17 + 25 + 83$

<div style="background:#f0fdf4;border-left:4px solid #22c55e;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$17 + 25 + 83 = 17 + 83 + 25$ (giao hoán)

$= (17 + 83) + 25$ (kết hợp)

$= 100 + 25 = 125$

</div>

b) $46 + 37 + 54 + 63$

<div style="background:#f0fdf4;border-left:4px solid #22c55e;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$46 + 37 + 54 + 63 = (46 + 54) + (37 + 63)$

$= 100 + 100 = 200$

</div>

<h3 id="phep-tru" style="margin-top:2rem;">📖 2. Phép trừ số tự nhiên</h3>

<div class="box-dinh-nghia">

**Định nghĩa:** Cho hai số tự nhiên $a$ và $b$ với $a \geq b$. Phép trừ $a$ cho $b$ cho ta số tự nhiên $c$ sao cho $b + c = a$, ký hiệu là $a - b = c$.

- $a$: **số bị trừ**
- $b$: **số trừ**
- $c = a - b$: **hiệu**

**Điều kiện:** $a \geq b$ (số bị trừ phải lớn hơn hoặc bằng số trừ)

</div>

**Tính chất của phép trừ:**

<div style="background:#fef3c7;border-left:4px solid #f59e0b;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

**1. Trừ đi số 0:**

$a - 0 = a$

Ví dụ: $50 - 0 = 50$

**2. Trừ đi chính nó:**

$a - a = 0$

Ví dụ: $30 - 30 = 0$

**3. Tính chất của phép trừ liên tiếp:**

$a - b - c = a - (b + c)$

Ví dụ: $100 - 20 - 30 = 100 - (20 + 30) = 100 - 50 = 50$

</div>

**Ví dụ 2:** Tính nhanh

a) $125 - 37 - 25$

<div style="background:#f0f9ff;border-left:4px solid #0ea5e9;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$125 - 37 - 25 = 125 - 25 - 37$

$= (125 - 25) - 37$

$= 100 - 37 = 63$

</div>

b) $200 - (45 + 55)$

<div style="background:#f0f9ff;border-left:4px solid #0ea5e9;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$200 - (45 + 55) = 200 - 100 = 100$

</div>

<h3 id="moi-lien-he" style="margin-top:2rem;">🔗 3. Mối liên hệ giữa phép cộng và phép trừ</h3>

<div class="box-dinh-ly">

**Mối liên hệ:**

Nếu $a + b = c$ thì:
- $c - a = b$
- $c - b = a$

**Ví dụ:** $5 + 7 = 12$ nên:
- $12 - 5 = 7$
- $12 - 7 = 5$

</div>

**Ứng dụng:** Tìm số chưa biết

**Ví dụ 3:** Tìm $x$ biết:

a) $x + 15 = 40$

<div style="background:#f0fdf4;border-left:4px solid #22c55e;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$x = 40 - 15 = 25$

**Thử lại:** $25 + 15 = 40$ ✓

</div>

b) $x - 23 = 17$

<div style="background:#f0fdf4;border-left:4px solid #22c55e;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$x = 17 + 23 = 40$

**Thử lại:** $40 - 23 = 17$ ✓

</div>

c) $56 - x = 28$

<div style="background:#f0fdf4;border-left:4px solid #22c55e;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$x = 56 - 28 = 28$

**Thử lại:** $56 - 28 = 28$ ✓

</div>

<h3 id="tinh-nhanh" style="margin-top:2rem;">⚡ 4. Các kỹ thuật tính nhanh</h3>

**Kỹ thuật 1:** Nhóm các số có tổng tròn chục, tròn trăm

<div style="background:#f0f9ff;border-left:4px solid #0ea5e9;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$27 + 48 + 73 + 52$

$= (27 + 73) + (48 + 52)$

$= 100 + 100 = 200$

</div>

**Kỹ thuật 2:** Thêm vào rồi bớt đi

<div style="background:#f0f9ff;border-left:4px solid #0ea5e9;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$47 + 99 = 47 + 100 - 1 = 147 - 1 = 146$

$85 - 49 = 85 - 50 + 1 = 35 + 1 = 36$

</div>

**Kỹ thuật 3:** Tách số để tính thuận tiện

<div style="background:#f0f9ff;border-left:4px solid #0ea5e9;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$156 - 28 - 56 = (156 - 56) - 28 = 100 - 28 = 72$

</div>

**📝 Bài toán 1:** Một cửa hàng sách có $250$ cuốn sách. Buổi sáng bán được $85$ cuốn, buổi chiều bán được $67$ cuốn.

a) Cả ngày cửa hàng bán được bao nhiêu cuốn sách?

b) Cuối ngày cửa hàng còn lại bao nhiêu cuốn sách?

<div style="background:#f0f9ff;border-left:4px solid #0ea5e9;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

**Giải:**

a) Số sách bán được cả ngày:

$85 + 67 = 152$ (cuốn)

b) Số sách còn lại:

$250 - 152 = 98$ (cuốn)

**Đáp số:** a) 152 cuốn; b) 98 cuốn

</div>

**📝 Bài toán 2:** Ba bạn An, Bình, Chi có tổng cộng $180$ viên bi. An có $65$ viên, Bình có $58$ viên.

a) Hai bạn An và Bình có tất cả bao nhiêu viên bi?

b) Chi có bao nhiêu viên bi?

<div style="background:#f0fdf4;border-left:4px solid #22c55e;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

**Giải:**

a) Số bi của An và Bình:

$65 + 58 = 123$ (viên)

b) Số bi của Chi:

$180 - 123 = 57$ (viên)

**Đáp số:** a) 123 viên; b) 57 viên

</div>

<GhiNho>

- **Phép cộng**: $a + b = c$ (tổng)
  - Giao hoán: $a + b = b + a$
  - Kết hợp: $(a + b) + c = a + (b + c)$
  - Cộng với 0: $a + 0 = a$
- **Phép trừ**: $a - b = c$ (hiệu), điều kiện $a \geq b$
  - $a - 0 = a$
  - $a - a = 0$
- **Mối liên hệ**: $a + b = c \Rightarrow c - a = b$ và $c - b = a$

</GhiNho>
