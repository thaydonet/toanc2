<KhamPha>

<h3 id="quy-tac">📖 1. Quy tắc chia đa thức cho đơn thức</h3>

<div class="box-dinh-ly">

**Quy tắc:** Chia từng hạng tử của đa thức cho đơn thức, rồi cộng các thương lại.

$(a + b) : c = a : c + b : c$ (với $c \\neq 0$)

</div>

**Điều kiện:** Đơn thức chia phải khác 0 và mỗi hạng tử của đa thức phải chia hết cho đơn thức.

<h3 id="vi-du" style="margin-top:2rem;">✏️ 2. Ví dụ minh họa</h3>

**Ví dụ 1:** $(6x^3 - 9x^2 + 3x) : 3x$

<div style="background:#f0f9ff;border-left:4px solid #0ea5e9;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

Chia từng hạng tử:

$= 6x^3 : 3x - 9x^2 : 3x + 3x : 3x$

$= 2x^2 - 3x + 1$

</div>

**Ví dụ 2:** $(15x^4y - 10x^3y^2 + 5x^2y) : 5x^2y$

<div style="background:#f0fdf4;border-left:4px solid #22c55e;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$= 15x^4y : 5x^2y - 10x^3y^2 : 5x^2y + 5x^2y : 5x^2y$

$= 3x^2 - 2xy + 1$

</div>

<h3 id="chu-y" style="margin-top:2rem;">⚠️ 3. Lưu ý quan trọng</h3>

<div class="box-nhan-xet">

1. Đơn thức chia phải khác 0
2. Mỗi hạng tử phải chia hết cho đơn thức
3. Khi chia: hệ số chia hệ số, số mũ trừ số mũ

</div>

**Ví dụ 3:** Kiểm tra phép chia

<div style="background:#fdf4ff;border-left:4px solid #a855f7;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$(x^3 + x) : x^2$ **không thực hiện được** vì $x : x^2 = x^{-1}$ (số mũ âm)

$(6x^2 - 4x) : 2x = 3x - 2$ ✓ (thực hiện được)

</div>

<h3 id="ung-dung" style="margin-top:2rem;">🔍 4. Ứng dụng</h3>

**Phân tích đa thức thành nhân tử:**

$6x^3 - 9x^2 + 3x = 3x(2x^2 - 3x + 1)$

Kiểm tra: $3x(2x^2 - 3x + 1) = 6x^3 - 9x^2 + 3x$ ✓

</KhamPha>

<h3 id="van-dung">🌍 Vận dụng</h3>

**📝 Bài toán:** Một hình chữ nhật có diện tích $S = 12x^3 - 8x^2$ (cm²) và chiều dài $4x$ (cm). Tính chiều rộng.

<div style="background:#f0f9ff;border-left:4px solid #0ea5e9;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

Chiều rộng $= S : $ chiều dài

$= (12x^3 - 8x^2) : 4x$

$= 3x^2 - 2x$ (cm)

- **Kiểm tra**: nhân ngược lại để kiểm tra
