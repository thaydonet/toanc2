<h3 id="cong-da-thuc">📖 1. Phép cộng đa thức</h3>

<div class="box-dinh-ly">

**Quy tắc:** Để cộng hai đa thức, ta:

1. Bỏ dấu ngoặc
2. Nhóm các hạng tử đồng dạng (cùng bậc)
3. Cộng các hạng tử đồng dạng
4. Thu gọn kết quả

</div>

**Ví dụ 1:** Cho $P(x) = 3x^2 - 2x + 5$ và $Q(x) = x^2 + 4x - 3$. Tính $P(x) + Q(x)$.

<div style="background:#f0f9ff;border-left:4px solid #0ea5e9;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$P(x) + Q(x) = (3x^2 - 2x + 5) + (x^2 + 4x - 3)$

$= 3x^2 - 2x + 5 + x^2 + 4x - 3$

$= (3x^2 + x^2) + (-2x + 4x) + (5 - 3)$

$= 4x^2 + 2x + 2$

</div>

<h3 id="tru-da-thuc" style="margin-top:2rem;">📖 2. Phép trừ đa thức</h3>

<div class="box-dinh-ly">

**Quy tắc:** Để trừ hai đa thức, ta:

1. Đổi dấu tất cả các hạng tử của đa thức bị trừ
2. Cộng đa thức thứ nhất với đa thức đối của đa thức thứ hai

$$P(x) - Q(x) = P(x) + (-Q(x))$$

</div>

**Ví dụ 2:** Cho $P(x) = 5x^2 - 3x + 1$ và $Q(x) = 2x^2 + x - 4$. Tính $P(x) - Q(x)$.

<div style="background:#f0f9ff;border-left:4px solid #0ea5e9;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$P(x) - Q(x) = (5x^2 - 3x + 1) - (2x^2 + x - 4)$

$= 5x^2 - 3x + 1 - 2x^2 - x + 4$

$= (5x^2 - 2x^2) + (-3x - x) + (1 + 4)$

$= 3x^2 - 4x + 5$

</div>

<h3 id="da-thuc-doi" style="margin-top:2rem;">📖 3. Đa thức đối</h3>

<div class="box-dinh-nghia">

**Đa thức đối** của $P(x)$ là $-P(x)$, được tạo bằng cách đổi dấu tất cả các hạng tử.

$$P(x) + (-P(x)) = 0$$

</div>

**Ví dụ 3:** Tìm đa thức đối của $P(x) = 3x^2 - 5x + 2$

<div style="background:#f0f9ff;border-left:4px solid #0ea5e9;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$-P(x) = -(3x^2 - 5x + 2) = -3x^2 + 5x - 2$

</div>

<h3 id="tinh-chat" style="margin-top:2rem;">📖 4. Tính chất</h3>

<div class="box-dinh-ly">

**Tính chất giao hoán:** $P(x) + Q(x) = Q(x) + P(x)$

**Tính chất kết hợp:** $[P(x) + Q(x)] + R(x) = P(x) + [Q(x) + R(x)]$

**Cộng với 0:** $P(x) + 0 = P(x)$

**Cộng với đa thức đối:** $P(x) + (-P(x)) = 0$

</div>

<h3 id="van-dung">🌍 Vận dụng</h3>

**Bài toán:** Một hình chữ nhật có chiều dài $P(x) = 2x + 3$ và chiều rộng $Q(x) = x + 1$. Tính chu vi hình chữ nhật.

<div style="background:#f0f9ff;border-left:4px solid #0ea5e9;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$= 2(3x + 4)$

$= 6x + 8$

</div>
