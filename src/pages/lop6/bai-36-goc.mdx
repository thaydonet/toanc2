<h3 id="so-sanh-goc" style="margin-top:2rem;">📖 4. So sánh góc</h3>

**Cách so sánh:**

1. **Dùng thước đo góc:** Đo số đo từng góc, rồi so sánh các số đo.

2. **Dùng compa:** Vẽ hai cung tròn có cùng bán kính từ đỉnh của mỗi góc, rồi so sánh độ dài cung.

**Ký hiệu so sánh:**
- ∠AOB = ∠COD (bằng nhau)
- ∠AOB < ∠COD (nhỏ hơn)
- ∠AOB > ∠COD (lớn hơn)

**Ví dụ:**

<div style="background:#f0f9ff;border-left:4px solid #0ea5e9;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

- ∠AOB = 45° (góc nhọn)
- ∠COD = 90° (góc vuông)
- ∠EOF = 120° (góc tù)

So sánh: ∠AOB < ∠COD < ∠EOF

</div>
