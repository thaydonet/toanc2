<KhamPha>

<h3 id="nhan-don-da">📖 1. Nhân đơn thức với đa thức</h3>

<div class="box-dinh-ly">

**Quy tắc:** Nhân đơn thức với từng hạng tử của đa thức, rồi cộng các tích lại.

$a(b + c) = ab + ac$

</div>

**Ví dụ 1:** $3x(2x^2 - 5x + 1) = 6x^3 - 15x^2 + 3x$

<h3 id="nhan-da-da" style="margin-top:2rem;">📖 2. Nhân đa thức với đa thức</h3>

<div class="box-dinh-ly">

**Quy tắc:** Nhân mỗi hạng tử của đa thức này với mỗi hạng tử của đa thức kia, rồi cộng các tích lại.

$(a + b)(c + d) = ac + ad + bc + bd$

</div>

**Ví dụ 2:** $(x + 2)(x + 3)$

<div style="background:#f0f9ff;border-left:4px solid #0ea5e9;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$= x \\cdot x + x \\cdot 3 + 2 \\cdot x + 2 \\cdot 3$

$= x^2 + 3x + 2x + 6$

$= x^2 + 5x + 6$

</div>

<h3 id="hang-dang-thuc" style="margin-top:2rem;">🔑 3. Hằng đẳng thức đáng nhớ</h3>

<div class="box-dinh-ly">

1. $(a + b)^2 = a^2 + 2ab + b^2$
2. $(a - b)^2 = a^2 - 2ab + b^2$
3. $(a + b)(a - b) = a^2 - b^2$

</div>

**Ví dụ 3:**

$(x + 3)^2 = x^2 + 6x + 9$

$(2x - 1)^2 = 4x^2 - 4x + 1$

$(x + 5)(x - 5) = x^2 - 25$

</KhamPha>
