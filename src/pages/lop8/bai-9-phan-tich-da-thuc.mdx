<KhamPha>

<h3 id="pp1">📖 1. Phương pháp đặt nhân tử chung</h3>

<div class="box-dinh-ly">

**Nguyên tắc:** Tìm nhân tử chung của các hạng tử và đặt ra ngoài dấu ngoặc.

$AB + AC = A(B + C)$

</div>

**Ví dụ 1:**

<div style="background:#f0f9ff;border-left:4px solid #0ea5e9;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$3x^2 + 6x = 3x(x + 2)$

$5x^3 - 10x^2 + 15x = 5x(x^2 - 2x + 3)$

</div>

<h3 id="pp2" style="margin-top:2rem;">📖 2. Dùng hằng đẳng thức</h3>

**Ví dụ 2:**

<div style="background:#f0fdf4;border-left:4px solid #22c55e;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$x^2 - 16 = (x - 4)(x + 4)$

$x^2 + 6x + 9 = (x + 3)^2$

$x^3 + 8 = (x + 2)(x^2 - 2x + 4)$

</div>

<h3 id="pp3" style="margin-top:2rem;">📖 3. Nhóm hạng tử</h3>

**Ví dụ 3:**

<div style="background:#fdf4ff;border-left:4px solid #a855f7;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$x^2 + 3x + 2x + 6 = (x^2 + 3x) + (2x + 6)$

$= x(x + 3) + 2(x + 3) = (x + 3)(x + 2)$

</div>

<h3 id="pp4" style="margin-top:2rem;">📖 4. Phối hợp các phương pháp</h3>

**Ví dụ 4:**

<div style="background:#fffbeb;border-left:4px solid #f59e0b;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$2x^2 - 8 = 2(x^2 - 4) = 2(x - 2)(x + 2)$

$x^3 - 4x = x(x^2 - 4) = x(x - 2)(x + 2)$

</div>

<h3 id="pp5" style="margin-top:2rem;">📖 5. Tách hạng tử</h3>

**Ví dụ 5:** Phân tích $x^2 + 5x + 6$

<div style="background:#f0f9ff;border-left:4px solid #0ea5e9;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

Tách $5x = 2x + 3x$:

$x^2 + 5x + 6 = x^2 + 2x + 3x + 6$

$= x(x + 2) + 3(x + 2) = (x + 2)(x + 3)$

</div>

</KhamPha>
