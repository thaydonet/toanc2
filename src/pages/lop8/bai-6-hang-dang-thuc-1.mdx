<KhamPha>

<h3 id="hdt-1">📖 1. Bình phương của một tổng</h3>

<div class="box-dinh-ly">

**Hằng đẳng thức 1:**

$(a + b)^2 = a^2 + 2ab + b^2$

**Phát biểu:** Bình phương của một tổng bằng bình phương số hạng thứ nhất cộng hai lần tích hai số hạng cộng bình phương số hạng thứ hai.

</div>

**Chứng minh:**

$(a + b)^2 = (a + b)(a + b) = a^2 + ab + ba + b^2 = a^2 + 2ab + b^2$

**Ví dụ 1:**

<div style="background:#f0f9ff;border-left:4px solid #0ea5e9;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$(x + 3)^2 = x^2 + 2 \\cdot x \\cdot 3 + 3^2 = x^2 + 6x + 9$

$(2x + 5)^2 = (2x)^2 + 2 \\cdot 2x \\cdot 5 + 5^2 = 4x^2 + 20x + 25$

</div>

**Ứng dụng tính nhanh:**

$101^2 = (100 + 1)^2 = 100^2 + 2 \\cdot 100 \\cdot 1 + 1^2 = 10000 + 200 + 1 = 10201$

<h3 id="hdt-2" style="margin-top:2rem;">📖 2. Bình phương của một hiệu</h3>

<div class="box-dinh-ly">

**Hằng đẳng thức 2:**

$(a - b)^2 = a^2 - 2ab + b^2$

**Phát biểu:** Bình phương của một hiệu bằng bình phương số hạng thứ nhất trừ hai lần tích hai số hạng cộng bình phương số hạng thứ hai.

</div>

**Ví dụ 2:**

<div style="background:#f0fdf4;border-left:4px solid #22c55e;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$(x - 4)^2 = x^2 - 2 \\cdot x \\cdot 4 + 4^2 = x^2 - 8x + 16$

$(3x - 2)^2 = (3x)^2 - 2 \\cdot 3x \\cdot 2 + 2^2 = 9x^2 - 12x + 4$

</div>

**Ứng dụng tính nhanh:**

$99^2 = (100 - 1)^2 = 100^2 - 2 \\cdot 100 \\cdot 1 + 1^2 = 10000 - 200 + 1 = 9801$

<h3 id="hdt-3" style="margin-top:2rem;">📖 3. Hiệu hai bình phương</h3>

<div class="box-dinh-ly">

**Hằng đẳng thức 3:**

$a^2 - b^2 = (a - b)(a + b)$

**Phát biểu:** Hiệu hai bình phương bằng tích của hiệu và tổng hai số hạng.

</div>

**Chứng minh:**

$(a - b)(a + b) = a^2 + ab - ba - b^2 = a^2 - b^2$

**Ví dụ 3:**

<div style="background:#fdf4ff;border-left:4px solid #a855f7;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$x^2 - 9 = x^2 - 3^2 = (x - 3)(x + 3)$

$4x^2 - 25 = (2x)^2 - 5^2 = (2x - 5)(2x + 5)$

</div>

**Ứng dụng tính nhanh:**

$103 \\times 97 = (100 + 3)(100 - 3) = 100^2 - 3^2 = 10000 - 9 = 9991$

<h3 id="ung-dung" style="margin-top:2rem;">🔧 4. Ứng dụng</h3>

**a) Rút gọn biểu thức:**

**Ví dụ 4:** Rút gọn $(x + 5)^2 - (x - 5)^2$

<div style="background:#fffbeb;border-left:4px solid #f59e0b;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$= (x^2 + 10x + 25) - (x^2 - 10x + 25)$

$= x^2 + 10x + 25 - x^2 + 10x - 25$

$= 20x$

**Cách khác:** Dùng $a^2 - b^2 = (a - b)(a + b)$

$= [(x + 5) - (x - 5)][(x + 5) + (x - 5)]$

$= [10][2x] = 20x$

</div>

**b) Chứng minh đẳng thức:**

**Ví dụ 5:** Chứng minh $(a + b)^2 = (a - b)^2 + 4ab$

<div style="background:#f0f9ff;border-left:4px solid #0ea5e9;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

VT $= a^2 + 2ab + b^2$

VP $= (a^2 - 2ab + b^2) + 4ab = a^2 + 2ab + b^2$

Vậy VT = VP (đpcm)

</div>
