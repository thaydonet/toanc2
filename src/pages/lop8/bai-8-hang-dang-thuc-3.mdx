<h3 id="hdt-6">📖 1. Tổng hai lập phương</h3>

<div class="box-dinh-ly">

**Hằng đẳng thức 6:**

$a^3 + b^3 = (a + b)(a^2 - ab + b^2)$

</div>

**Ví dụ 1:**

<div style="background:#f0f9ff;border-left:4px solid #0ea5e9;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$x^3 + 8 = x^3 + 2^3 = (x + 2)(x^2 - 2x + 4)$

$27x^3 + 1 = (3x)^3 + 1^3 = (3x + 1)(9x^2 - 3x + 1)$

</div>

<h3 id="hdt-7" style="margin-top:2rem;">📖 2. Hiệu hai lập phương</h3>

<div class="box-dinh-ly">

**Hằng đẳng thức 7:**

$a^3 - b^3 = (a - b)(a^2 + ab + b^2)$

</div>

**Ví dụ 2:**

<div style="background:#f0fdf4;border-left:4px solid #22c55e;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$x^3 - 27 = x^3 - 3^3 = (x - 3)(x^2 + 3x + 9)$

$8x^3 - 125 = (2x)^3 - 5^3 = (2x - 5)(4x^2 + 10x + 25)$

</div>

<h3 id="ung-dung" style="margin-top:2rem;">💡 3. Ứng dụng tính nhanh</h3>

**Ví dụ 3:** Tính $7^3 + 3^3$

<div style="background:#fdf4ff;border-left:4px solid #a855f7;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$7^3 + 3^3 = (7 + 3)(7^2 - 7 \\cdot 3 + 3^2)$

$= 10(49 - 21 + 9) = 10 \\times 37 = 370$

</div>
