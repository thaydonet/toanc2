<h3 id="mo-dau">🍕 Cộng phần bánh</h3>

Lan mua $\dfrac{1}{4}$ cái bánh pizza, sau đó mua thêm $\dfrac{2}{4}$ cái bánh nữa. Hỏi Lan mua tất cả bao nhiêu phần bánh?

<div style="display:flex; gap:2rem; align-items:center; flex-wrap:wrap; justify-content:center; margin:1.5rem 0;">
  <div style="text-align:center;">
    <p>$\dfrac{1}{4}$</p>
  </div>
  <div style="text-align:center;font-size:1.5rem;">+</div>
  <div style="text-align:center;">
    <p>$\dfrac{2}{4}$</p>
  </div>
  <div style="text-align:center;font-size:1.5rem;">=</div>
  <div style="text-align:center;">
    <p>$\dfrac{3}{4}$</p>
  </div>
</div>

**Tổng: $\dfrac{1}{4} + \dfrac{2}{4} = \dfrac{3}{4}$**

Nhưng nếu Lan mua $\dfrac{1}{3}$ và $\dfrac{1}{4}$ thì sao? Hãy học cách cộng các phân số khác mẫu!

<KhamPha title="Khám phá">

<h3 id="cong-cung-mau" style="margin-top:2rem;">📖 1. Cộng phân số cùng mẫu</h3>

<div class="box-dinh-ly">

$$\frac{a}{m} + \frac{b}{m} = \frac{a + b}{m} \quad (m > 0)$$

**Quy tắc:** Cộng các tử số, giữ nguyên mẫu số.

</div>

**Ví dụ 1:** $\dfrac{2}{9} + \dfrac{5}{9} = \dfrac{2 + 5}{9} = \dfrac{7}{9}$

<h3 id="cong-khac-mau" style="margin-top:2rem;">📖 2. Cộng phân số khác mẫu</h3>

<div class="box-dinh-ly">

**Bước 1:** Quy đồng mẫu số các phân số

**Bước 2:** Cộng các phân số cùng mẫu

</div>

**Ví dụ 2:** Tính $\dfrac{1}{3} + \dfrac{1}{4}$

- BCNN(3, 4) = 12
- $\dfrac{1}{3} = \dfrac{4}{12}$
- $\dfrac{1}{4} = \dfrac{3}{12}$
- $\dfrac{4}{12} + \dfrac{3}{12} = \dfrac{7}{12}$

<FractionCalc client:load operation="add" />

<h3 id="tru-phan-so" style="margin-top:2rem;">📖 3. Trừ phân số</h3>

<div class="box-dinh-ly">

$$\frac{a}{m} - \frac{b}{m} = \frac{a - b}{m} \quad (m > 0)$$

**Quy tắc:** Trừ các tử số, giữ nguyên mẫu số (các phân số phải cùng mẫu).

</div>

**Ví dụ 3:** Tính $\dfrac{7}{10} - \dfrac{2}{5}$

- $\dfrac{2}{5} = \dfrac{4}{10}$
- $\dfrac{7}{10} - \dfrac{4}{10} = \dfrac{3}{10}$

<FractionCalc client:load operation="subtract" />

<h3 id="cong-tru-phan-am" style="margin-top:2rem;">📖 4. Cộng, trừ phân số âm</h3>

<div class="box-luu-y">

**Quy tắc cộng phân số có dấu:**

1. Cùng dấu: cộng giá trị tuyệt đối, giữ ngấu chung
2. Khác dấu: trừ giá trị tuyệt đối (lớn trừ nhỏ), lấy dấu của số có GTTĐ lớn hơn

$$\frac{-a}{b} + \frac{c}{d} = \frac{-ad + bc}{bd}$$

</div>

**Ví dụ 4a:** $\dfrac{-2}{5} + \dfrac{-3}{5} = \dfrac{-5}{5} = -1$

**Ví dụ 4b:** $\dfrac{-2}{5} + \dfrac{3}{5} = \dfrac{1}{5}$

**Ví dụ 4c:** $\dfrac{3}{5} + \dfrac{-7}{5} = \dfrac{-4}{5}$

<h3 id="tinh-chat" style="margin-top:2rem;">📖 5. Tính chất của phép cộng phân số</h3>

<div class="box-dinh-ly">

Phép cộng phân số có các tính chất:

1. **Giao hoán:** $\dfrac{a}{b} + \dfrac{c}{d} = \dfrac{c}{d} + \dfrac{a}{b}$

2. **Kết hợp:** $\left(\dfrac{a}{b} + \dfrac{c}{d}\right) + \dfrac{e}{f} = \dfrac{a}{b} + \left(\dfrac{c}{d} + \dfrac{e}{f}\right)$

3. **Cộng với 0:** $\dfrac{a}{b} + 0 = \dfrac{a}{b}$

</div>

**Ví dụ 5:** Tính nhanh: $\dfrac{1}{2} + \dfrac{3}{7} + \dfrac{1}{2} + \dfrac{4}{7}$

$\left(\dfrac{1}{2} + \dfrac{1}{2}\right) + \left(\dfrac{3}{7} + \dfrac{4}{7}\right) = 1 + 1 = 2$

<h3 id="phep-tinh-lien-tiep" style="margin-top:2rem;">📖 6. Dãy phép tính cộng trừ phân số</h3>

<div class="box-luu-y">

Thực hiện từ **trái sang phải**, hoặc theo thứ tự:
1. Trong ngoặc trước
2. Cộng trừ liên tiếp

</div>

**Ví dụ 6:** Tính $\dfrac{3}{4} - \dfrac{1}{4} + \dfrac{2}{4}$

$\dfrac{3 - 1 + 2}{4} = \dfrac{4}{4} = 1$

<div style="background:#f0fdf4;border-left:4px solid #22c55e;border-radius:8px;padding:1rem 1.25rem;margin:1rem 0;">

$\dfrac{3}{5} = \dfrac{12}{20}$, $\dfrac{1}{4} = \dfrac{5}{20}$

<GhiNho title="Ghi nhớ">

<div class="box-tong-hop" style="background:#faf5ff;border-left:4px solid #9333ea;border-radius:8px;padding:1.5rem;margin:1rem 0;">

**Tổng kết bài học:**

**1. Cộng phân số:**
- Cùng mẫu: $\dfrac{a}{m} + \dfrac{b}{m} = \dfrac{a+b}{m}$
- Khác mẫu: Quy đồng mẫu rồi cộng

**2. Trừ phân số:**
- Cùng mẫu: $\dfrac{a}{m} - \dfrac{b}{m} = \dfrac{a-b}{m}$
- Khác mẫu: Quy đồng mẫu rồi trừ

**3. Phân số âm:**
- Cùng dấu: Cộng GTTĐ, giữ dấu chung
- Khác dấu: Trừ GTTĐ, lấy dấu của số lớn hơn

**4. Tính chất phép cộng:**
- Giao hoán: $a + b = b + a$
- Kết hợp: $(a + b) + c = a + (b + c)$
- Cộng với 0: $a + 0 = a$
