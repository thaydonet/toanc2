<h3 id="phan-so-thap-phan" style="margin-top:2rem;">📖 1. Phân số thập phân và số thập phân</h3>

<div class="box-dinh-nghia">

**Phân số thập phân** là phân số có mẫu là $10^n$ ($n$ là số tự nhiên).

**Số thập phân** gồm hai phần:
- **Phần nguyên** (trước dấu phẩy)
- **Phần thập phân** (sau dấu phẩy)

</div>

**Cách chuyển đổi:**

| Phân số thập phân | Số thập phân |
|------------------|--------------|
| $\frac{17}{10}$ | 1,7 |
| $\frac{34}{100}$ | 0,34 |
| $\frac{25}{1000}$ | 0,025 |
| $\frac{-7}{100}$ | -0,07 |

**Quy tắc:** Mỗi chữ số ở mẫu tương ứng với một chữ số ở phần thập phân.

**Ví dụ 1:** Viết $\frac{798}{10}$ dưới dạng số thập phân

$\frac{798}{10} = 79,8$ (dịch dấu phẩy sang trái 1 chữ số)

<h3 id="so-doi" style="margin-top:2rem;">📖 2. Số đối của số thập phân</h3>

<div class="box-dinh-nghia">

**Số đối** của số thập phân $a$ là số $-a$ sao cho $a + (-a) = 0$.

</div>

**Ví dụ 2:**
- Số đối của $3,5$ là $-3,5$
- Số đối của $-2,17$ là $2,17$
- Số đối của $0$ là $0$

<h3 id="so-sanh" style="margin-top:2rem;">📖 3. So sánh hai số thập phân</h3>

<div class="box-dinh-ly">

**Quy tắc so sánh:**
1. Số thập phân dương > số thập phân âm
2. Số thập phân âm nhỏ hơn 0
3. So sánh phần nguyên trước
4. Nếu phần nguyên bằng nhau, so sánh phần thập phân (thêm chữ số 0 nếu cần)

</div>

**Ví dụ 3:** So sánh 3,14 và 3,2

- Phần nguyên: $3 = 3$
- Phần thập phân: $14 < 20$ (so sánh $0,14$ và $0,2$)
- Vậy $3,14 < 3,2$

**Ví dụ 4:** Sắp xếp: $-8,9$; $-8,152$; $0$; $0,12$ theo thứ tự tăng dần

$-8,9 < -8,152 < 0 < 0,12$

<h3 id="vi-du-thuc-te" style="margin-top:2rem;">📖 4. Ứng dụng thực tế</h3>

Số thập phân được sử dụng rộng rãi trong đo lường:

| Đại lượng | Ví dụ số thập phân |
|-----------|---------------------|
| Chiều cao | 1,65 m |
| Cân nặng | 45,5 kg |
| Nhiệt độ | 36,5°C |
| Tiền | 125 000,00 đồng |

**Tổng kết bài học:**

**1. Phân số thập phân và số thập phân**
- Phân số thập phân: mẫu là $10^n$
- Số thập phân gồm phần nguyên + phần thập phân (ngăn cách bởi dấu phẩy)

**2. Chuyển đổi**
- $\frac{a}{10^n}$ = chuyển dấu phẩy sang trái $n$ vị trí
- Số thập phân $a,bcd$ = $\frac{abcd}{10^n}$ ($n$ là số chữ số phần thập phân)

**3. Số đối**
- Số đối của $a$ là $-a$
- $a + (-a) = 0$

**4. So sánh số thập phân**
- Dương > 0 > Âm
- So sánh phần nguyên trước
- Nếu bằng nhau, so sánh phần thập phân
